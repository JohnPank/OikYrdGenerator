Файл с исходными данными должен иметь такой формат:

Ludkovo
7
#TC2:7:103
#TC2:7:104
#TC2:7:105
#TC2:7:201
#TC2:7:301
#TC2:7:401
#TC2:7:501
8
#TC2:8:103
#TC2:8:201
9
#TC2:9:103
#TC2:9:104
#TC2:9:105
#TC2:9:106

название подстанции пишется транслитералом.
цифры указывают номера ВЛ.
индикатор несоответствия всегда имеет номер 200.

Исходный файл должен иметь расширение .txt, а результат сохраняетса в файле с расширением .clc.
При вводе имени файла расширение указывать не нужно!
